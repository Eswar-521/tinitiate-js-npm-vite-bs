# JS-NPM-VITE-BS Project Documentation

## Project Overview

This project is built using **JavaScript, Vite, Bootstrap, and React**. It includes a structured component-based architecture with anthentication pages.

## Installation & Setup

### Prerequisites 

 Ensure you have the following installed :
-Node.js (Recommended : **18.x or 20.x**)
-Yarn Package Manager

### Install Yarn Globally
```bash
npm install -g yarn
```


## Create a Vite Project with Yarn 
```bash
yarn create vite my-yarnvite
```

## Install Dependencies 
```bash 
yarn add react-router-dom bootstrap react-bootstrap
```

## Start the Development Server 
```bash
yarn dev
```
  
## Running the project

#### Start the development server using:
```bash
Yarn dev
```

#### This will lanch the app at [link](https://localhost:5173)
